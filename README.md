**BudgetFlow - Smart Expense Tracker**

BudgetFlow is a modern, lightweight web-based expense tracker designed to help users manage income and expenses efficiently. It provides an interactive dashboard with live charts, smooth animations, and a user-friendly interface to help users visualize and control their finances.

**Features**
* Add and track income and expenses with description and category.
* Live chart visualization of income vs. expenses.
* Responsive design for desktop and mobile.
* Clear all transactions option to reset records.
* Modern, animated contact form.
* Local storage support for persistent data.

**Technologies Used**
* HTML5 for structure.
* CSS3 for styling, animations, and responsiveness.
* JavaScript (ES6) for transaction handling and chart updates.
* Chart.js for dynamic chart visualization.
* Normalize.css for consistent cross-browser styling.
* FontAwesome for icons.

**Installation**
* Clone the repository:
* git clone https://github.com/chavergara18/Web-System-and-Technology.git
* Navigate to the project folder:
cd BudgetFlow
* Open index.html in a web browser.

**Usage**
* Navigate to the Home section to learn about the project.
* Scroll or click the "Get Started" button to access the Tracker
* Add a transaction by entering description, amount, and type (Income/Expense).
* View the live chart updating automatically.
* Use "Clear All" to reset transactions.
* Use the Contact section to send a message (frontend alert only).
